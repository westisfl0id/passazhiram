# Пассажирам.РФ

Веб-приложение для записи пользователей на курсы обучения водителей пассажирского транспорта. Проект разработан в рамках подготовки к демонстрационному экзамену по специальности **09.02.07 «Информационные системы и программирование»**.

## Возможности системы

### Пользователь

* Регистрация нового пользователя.
* Авторизация по логину и паролю.
* Просмотр личного кабинета.
* Создание заявок на обучение.
* Просмотр истории заявок.
* Оставление отзывов после завершения обучения.

### Администратор

* Просмотр всех заявок.
* Изменение статуса заявок:

    * Новая
    * Идет обучение
    * Обучение завершено
* Управление заявками пользователей.

## Запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/westisfl0id/passazhiram.git
```

### 2. Создать файл

```
src/main/resources/application.properties
```

на основе

```
application.properties.example
```

### 3. Заполнить настройки базы данных

```properties
spring.datasource.url=jdbc:postgresql://localhost:5440/YOUR_DATABASE
spring.datasource.username=YOUR_USERNAME
spring.datasource.password=YOUR_PASSWORD
```

### 4. Запустить PostgreSQL

Создать базу данных и убедиться, что сервер PostgreSQL запущен.

### 5. Запустить приложение

```
PassagiramApplication
```

или выполнить

```bash
mvn spring-boot:run
```
